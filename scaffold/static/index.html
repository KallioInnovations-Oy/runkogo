// what you can build

REST APIs

JSON request/response helpers, pagination, validation errors, and consistent error shapes out of the box.

Microservice clusters

Service-to-service client with retries, circuit breaking, and automatic request ID / trace ID propagation.

SPA & mobile backends

CORS, rate limiting, security headers, and Bearer token auth patterns ready for frontend consumption.

Internal tools

Health checks for orchestrators, structured logging for observability, and env-based config for 12-factor deploys.

// framework modules

app.go

Lifecycle: startup hooks → serve → signal → drain → shutdown hooks. TLS support, HOST/PORT config, port validation.

router.go

Go 1.22 ServeMux with route groups and middleware chains. app.Router.Group("/api", auth)

middleware.go

Recovery, body limit, request ID, client IP, logger with query redaction, CORS, rate limiting.

security.go

Security headers: nosniff, frame deny, cache-control, permissions policy, optional HSTS.

proxy.go

Trusted proxy resolver. Right-to-left XFF walking. Secure by default — headers ignored without config.

client.go

Service-to-service HTTP with retries, exponential backoff, circuit breaker, header propagation.

response.go

JSON, errors, pagination, body decoding with trailing data check and size limits.

hosts.go

Allowed host validation. Prevents host header injection for routing, caching, and password reset attacks.

logger.go

Structured JSON logging via slog. Context-aware: automatically includes request ID, client IP, user ID.

config.go

Typed env vars: MustGet, GetInt, GetBool, GetDuration, GetSlice. Fail fast.

context.go

Request-scoped values: request ID, client IP, user ID, trace ID. Flows through every function call.

sanitize.go

Input sanitization. External IDs validated: [a-zA-Z0-9_-] max 64 chars. Rejects injection attempts.

// quick start

package main

import (
    "time"
    runko "github.com/kallioinnovations/runkogo"
)

func main() {
    app := runko.New(runko.Options{
        ServiceName: "my-service",
    })

    // Global middleware — applied to every request.
    app.Router.Use(
        runko.Recovery(app.Logger),
        runko.BodyLimit(1<<20),
        runko.DefaultSecurityHeaders(),
        runko.RequestIDMiddleware(),
        runko.ClientIPMiddleware(app.Proxy),
        runko.Logger(app.Logger),
    )

    // Routes.
    app.Router.HandleFunc("GET /hello", func(w, r) {
        runko.JSON(w, 200, runko.Map{"msg": "hello"})
    })

    // Protected group.
    api := app.Router.Group("/api/v1", authMiddleware)
    api.HandleFunc("GET /users/{id}", getUser)

    app.Run() // Blocks until SIGINT/SIGTERM.
}

// security conventions

CONV-01
Secure by default. X-Forwarded-For ignored unless TrustedProxies configured. CORS denied unless origins listed.
CONV-02
Sanitize external input. Request IDs validated: [a-zA-Z0-9_-] max 64 chars. Injection attempts silently replaced.
CONV-03
No internal details exposed. Health checks show pass/fail only. Error messages never contain hostnames or connection strings.
CONV-04
Security headers on every response. nosniff, frame deny, cache-control, permissions-policy. Inspect any response above.
CONV-05
Fail fast on danger. CORS wildcard + credentials → panic. Invalid proxy CIDR → panic. Partial TLS config → panic.
CONV-06
Bounded resources. Body: 1MB. Rate limiter: 10K clients. Service responses: 10MB. Request IDs: 64 chars.
CONV-07
Timeout everything. Server read/write/idle timeouts. Per-request service client timeout. Per-check health timeouts.
CONV-08
Audit trail. Every request logged: method, path, status, duration, request ID, resolved client IP.

// deploy

# Docker (multi-stage, scratch base)
FROM golang:1.22-alpine AS build
WORKDIR /app
COPY . .
RUN CGO_ENABLED=0 go build -o /service .

FROM scratch
COPY --from=build /service /service
EXPOSE 19100
ENTRYPOINT ["/service"]

# Result: ~8MB container, starts in milliseconds
# Environment variables
PORT=19100          # Server port (default: 19100)
HOST=0.0.0.0        # Bind address (default: 0.0.0.0)
LOG_LEVEL=info       # debug, info, warn, error
HEALTH_DETAIL=false  # Show error details in /readyz
